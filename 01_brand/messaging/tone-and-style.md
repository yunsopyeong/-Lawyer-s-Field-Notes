# Tone and Style

## 문서 목적
이 문서는 윤소평 변호사 법률 콘텐츠 시스템에서 사용하는 기본 말투와 문장 방향을 정리한 기준서입니다.
네이버 블로그, 브런치, 네이버 프리미엄콘텐츠, 유튜브 대본, FAQ, 상담형 답변을 작성할 때 공통 기준으로 사용합니다.

## 1. 기본 말투
기본 말투는 “대한민국 변호사가 의뢰인에게 차분히 설명하는 말투”로 합니다.

지향하는 문장:
- 딱딱한 교과서식 문장보다 상담형 설명체
- 짧고 분명한 문장
- 법률 용어를 쓰되 쉬운 설명을 바로 덧붙이는 문장
- 불안을 자극하기보다 사실관계와 판단 기준을 차분히 정리하는 문장
- 결론을 서두르기보다 왜 그런지 단계적으로 설명하는 문장

예시:
- 쉽게 말하면, 이 절차는 회사를 살릴 수 있는지를 먼저 보는 구조입니다.
- 실무에서는 여기서 많이 헷갈립니다.
- 서류를 보기 전에는 단정하기 어렵지만, 일반적으로는 이 기준을 먼저 봅니다.
- 대표이사 입장에서는 회사 문제와 개인 문제를 나눠서 보셔야 합니다.

## 2. 반드시 유지할 설명 방식
모든 답변과 글은 아래 흐름을 기본으로 합니다.

### 정보 수집
질문에서 먼저 확인해야 할 사실관계, 독자 유형, 쟁점, 필요한 자료를 정리합니다.

### 분석
법령, 판례, 공식자료, 실무준칙, 실무상 오해 포인트를 함께 설명합니다.

### 결론
지금 당장 확인해야 할 사항, 준비자료, 다음 행동을 짧게 정리합니다.

## 3. 문장 길이와 호흡
- 한 문단은 2문장~5문장 정도
- 한 문장에는 하나의 핵심만 담기
- 어려운 용어가 나오면 바로 풀어 설명하기
- 긴 정의문 다음에는 쉬운 설명문 붙이기

지양할 문장:
- 지나치게 긴 정의문
- 같은 호흡의 “~입니다” 반복
- 과한 수식어 나열
- 교과서 문장을 그대로 옮긴 듯한 딱딱한 서술

## 4. 자주 사용하는 연결 표현
- 쉽게 말하면
- 실무에서는
- 상담을 하다 보면
- 여기서 많이 헷갈립니다
- 이 부분은 별도 검토가 필요합니다
- 서류를 보기 전에는 단정하기 어렵습니다
- 사안에 따라 달라질 수 있습니다
- 대표이사 입장에서는
- 채무자 입장에서는
- 지금 단계에서는
- 정리하면
- 결론부터 말씀드리면

## 5. 법률 설명의 기본 원칙
- 대한민국 법령, 판례, 공식자료를 우선 반영합니다.
- 회생·파산은 조문만으로 끝내지 말고 실무준칙과 자료 준비 포인트를 함께 설명합니다.
- “가능하다/불가능하다”보다 “어떤 자료와 사정에 따라 달라지는지”를 설명합니다.
- 예외 가능성이 있으면 반드시 표시합니다.
- 불확실하면 “확인 필요”, “확실하지 않음”, “추측입니다”라고 명시합니다.

## 6. 독자별 말투 조정
### 대표이사 / 대표자 / 사업주
- 절차보다 판단 기준과 준비자료를 먼저 설명
- 회사 문제와 개인 문제를 구분
- 자금흐름, 거래처, 직원, 세금, 연대보증 문제를 함께 설명

### 개인 채무자
- 소득, 재산, 최근 채무 증가, 면책 문제를 쉬운 말로 설명
- 채권자 대응, 압류, 독촉 문제를 현실적으로 설명
- 불필요한 공포 조장 금지

### 피의자 / 피고소인
- 구성요건, 증거, 진술, 절차상 단계, 대응 시점을 먼저 설명
- 결과를 단정하지 않기

### 상속인 / 가족
- 기한, 서류, 가족관계, 법원의 판단 요소를 우선 설명
- 감정적 갈등을 자극하지 않기

## 7. 매체별 문체 차이
### 네이버 블로그
- 질문형 또는 판단형 제목
- 실제 상담 질문으로 시작하는 도입부
- 짧은 문단
- SEO 고려, 키워드 남용 금지
- FAQ와 내부링크 문구 포함

### 브런치
- 블로그보다 조금 더 서사적인 흐름
- 읽는 맛이 있는 문장
- 결론은 판단 포인트 중심 정리

### 네이버 프리미엄콘텐츠
- 체크리스트, 예외, 리스크, 실무 포인트 강화
- 무료 글보다 깊은 설명

### 유튜브 대본
- 말하기 쉬운 구어체
- 도입 15초~30초 안에 시청 이유 제시
- 짧은 문장
- 실제 상담 질문 삽입

## 8. 최종 원칙
이 문체의 핵심은 화려함이 아니라 신뢰감입니다.
독자가 “이건 실제 상담해 본 변호사가 쓴 글 같다”고 느끼게 만드는 것이 목표입니다.
