# Naver Blog Main Prompt

## 문서 목적
이 프롬프트는 네이버 블로그 발행용 원고를 작성할 때 사용하는 메인 프롬프트입니다.

## 사용 목적
- 대한민국 법률에 맞는 정보형 블로그 글 작성
- 사람이 실제 상담하고 설명한 글처럼 자연스럽게 구성
- 네이버 블로그에 맞는 SEO형 구조 반영
- 과장 광고나 키워드 남용 없이 클릭률과 체류 시간을 함께 고려
- 정보 수집 → 분석 → 결론 구조 유지
- FAQ, 내부링크, 해시태그까지 한 번에 설계

## 입력 전 확인할 사항
- 주 키워드
- 보조 키워드
- 독자 유형
- 검색 의도
- 반드시 들어갈 법령
- 반드시 들어갈 판례 또는 판례 취지
- 반드시 들어갈 공식자료
- 실무상 꼭 넣을 포인트
- 실제 상담에서 자주 듣는 질문
- 내가 강조하고 싶은 결론
- 피하고 싶은 표현
- 내부링크로 연결할 글
- 참고 자료 원문

## 메인 프롬프트
```text
아래 자료를 바탕으로 네이버 블로그 발행용 원고를 작성해 주세요.

[핵심 목표]
- 대한민국 법령, 판례, 공식자료를 우선 반영해 신뢰도 높은 정보형 글로 작성
- 사람이 실제 상담하면서 설명한 글처럼 자연스럽게 작성
- 네이버 블로그에 맞게 SEO를 고려하되 키워드 과다반복, 복사문체, 과장표현은 피하기
- 독자가 실제로 궁금해하는 질문 순서대로 설명하기
- 글 전체는 정보 수집 → 분석 → 결론 구조를 유지하기

[문체 기준]
- 변호사가 의뢰인에게 차분히 설명하는 말투
- 짧고 분명한 문장
- 법률 용어를 쓰되 쉬운 설명을 바로 덧붙이기
- “실무에서는”, “상담을 하다 보면”, “여기서 많이 헷갈립니다” 같은 자연스러운 연결 표현 사용
- 공포 조장, 광고형 과장, 결과 단정 표현 금지

[반드시 반영할 사항]
1. 제목은 질문형 또는 판단형으로 5개 제안
2. 첫 문단은 실제 상담에서 자주 듣는 질문이나 상황으로 시작
3. 소제목은 정보형과 상담형을 섞어 구성
4. 본문에 실무상 자주 오해하는 부분을 최소 2곳 이상 포함
5. 법령, 판례, 공식자료를 문장 안에 자연스럽게 반영
6. FAQ는 실제 의뢰인이 묻는 말처럼 5개 작성
7. 결론은 광고문이 아니라 체크포인트 정리형으로 마무리
8. 내부링크 문구 3개와 해시태그 10개 제안

[출력 형식]
1. SEO 제목 5개
2. 도입부 2개
3. 최종 블로그 본문
4. FAQ 5개
5. 결론
6. 내부링크 문구 3개
7. 해시태그 10개

[추가 정보]
- 주 키워드:
- 보조 키워드:
- 독자 유형:
- 검색 의도:
- 반드시 들어갈 법령:
- 반드시 들어갈 판례 또는 판례 취지:
- 반드시 들어갈 공식자료:
- 실무상 꼭 넣을 포인트:
- 실제 상담에서 자주 듣는 질문:
- 내가 강조하고 싶은 결론:
- 피하고 싶은 표현:
- 내부링크로 연결할 글:
- 참고 자료 원문:

[원고 또는 참고 자료]
여기에 붙여넣기
```

## 사용 예시
### 예시 1
- 주 키워드: 법인파산 절차
- 보조 키워드: 기업파산 기간, 대표이사 책임, 법인파산 비용
- 독자 유형: 대표이사
- 검색 의도: 폐업과 파산의 차이, 실제 신청 필요성 판단

### 예시 2
- 주 키워드: 개인회생 신청자격
- 보조 키워드: 개인회생 소득, 재산, 최근 대출
- 독자 유형: 개인 채무자
- 검색 의도: 개인회생 가능 여부 판단
