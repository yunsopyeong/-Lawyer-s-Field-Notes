# YouTube Script Main Prompt

## 문서 목적
이 프롬프트는 윤소평 변호사 유튜브 채널용 설명형 대본을 만들기 위한 메인 프롬프트입니다.

## 사용 목적
- 대한민국 법률에 맞는 정확한 유튜브 대본 작성
- 변호사가 카메라 앞에서 차분히 설명하는 말투 유지
- 초반 15초~30초 안에 시청자가 끝까지 볼 이유 제시
- 실제 상담 질문을 영상 구조에 반영
- 실무상 오해 포인트와 체크포인트를 분명하게 정리
- 제목, 썸네일, 설명란까지 한 번에 설계

## 입력 전 확인할 사항
- 주제
- 핵심 키워드
- 보조 키워드
- 예상 시청자
- 영상 목적
- 반드시 들어갈 법령
- 반드시 들어갈 공식자료 또는 판례 취지
- 실무상 꼭 넣을 포인트
- 실제 상담에서 자주 나오는 질문
- 강조하고 싶은 결론
- 피하고 싶은 표현
- 참고 자료 원문

## 메인 프롬프트
```text
아래 자료를 바탕으로 유튜브 촬영용 설명형 대본을 작성해 주세요.

[핵심 목표]
- 대한민국 법령, 판례, 공식자료를 바탕으로 정확하게 설명할 것
- 변호사가 의뢰인이나 시청자에게 차분히 설명하는 말투를 유지할 것
- 처음 15초~30초 안에 시청자가 왜 끝까지 봐야 하는지 드러나게 구성할 것
- 실제 상담에서 자주 받는 질문을 도입부와 본문에 반영할 것
- 정보 수집 → 분석 → 결론 흐름을 영상 대본 구조에 맞게 적용할 것

[문체 기준]
- 말하기 쉬운 짧은 문장
- 구어체이되 법률 용어는 정확하게 설명
- 과장된 방송 톤 금지
- 공포 조장, 광고형 단정 표현 금지

[반드시 반영할 사항]
1. 영상 제목 5개
2. 썸네일 문구 10개
3. 오프닝 훅 3개
4. 전체 대본은 도입 → 핵심 설명 → 실무상 오해 → 정리 순서로 작성
5. 실제 상담에서 자주 받는 질문 2개 이상 포함
6. 실무상 자주 오해하는 부분 2곳 이상 포함
7. 법령, 판례, 공식자료를 너무 딱딱하지 않게 풀어서 설명
8. 영상 말미에는 무리한 상담 유도 대신 다음 행동 2개 제시
9. 설명란 초안 포함

[출력 형식]
1. 제목 5개
2. 썸네일 문구 10개
3. 오프닝 훅 3개
4. 전체 대본
5. 클로징 문구 3개
6. 설명란 초안

[추가 정보]
- 주제:
- 핵심 키워드:
- 보조 키워드:
- 예상 시청자:
- 영상 목적:
- 반드시 들어갈 법령:
- 반드시 들어갈 판례 또는 판례 취지:
- 반드시 들어갈 공식자료:
- 실무상 꼭 넣을 포인트:
- 실제 상담에서 자주 듣는 질문:
- 강조하고 싶은 결론:
- 피하고 싶은 표현:
- 참고 자료 원문:

[자료]
여기에 붙여넣기
```

## 유튜브 대본 구조 기준
### 1. 도입
- 질문 하나로 시작
- 왜 이 주제가 중요한지 바로 제시
- 오늘 영상에서 무엇을 정리할지 짧게 예고

### 2. 핵심 설명
- 개념 설명
- 절차 또는 판단 기준 설명
- 시청자가 가장 궁금해하는 순서대로 설명

### 3. 실무상 오해
- 자주 틀리는 부분 2개 이상 분리
- 실제 상담형 문장 삽입

### 4. 정리
- 오늘 핵심 2~3줄 정리
- 지금 당장 확인할 체크포인트 제시
- 다음 행동 1~2개 안내
