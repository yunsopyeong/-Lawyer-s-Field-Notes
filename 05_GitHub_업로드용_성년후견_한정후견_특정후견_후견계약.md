# 성년후견·한정후견·특정후견·후견계약 구별방법

## 문서 목적
이 문서는 윤소평 변호사 콘텐츠 시스템에서 성년후견, 한정후견, 특정후견, 후견계약(임의후견) 주제를 다룰 때 사용하는 GitHub 업로드용 주제 요약 파일이다.

## 1. 핵심 메시지
후견제도는 하나가 아니다. 성년후견은 가장 넓고 지속적인 보호, 한정후견은 중요한 사무 전반에 대한 상당한 지원, 특정후견은 특정 사무 또는 일시적 후원, 후견계약은 장래를 대비한 사전 설계형 제도다.

## 2. 법적 근거
- 민법 제9조: 성년후견
- 민법 제12조: 한정후견
- 민법 제14조의2: 특정후견
- 민법 제959조의14: 후견계약(임의후견)
- 민법 제959조의20: 후견계약과 성년후견·한정후견·특정후견의 관계
- 가사소송법 제45조의2: 정신상태의 감정 등

## 3. 제도별 구별 포인트
### 성년후견
- 전반적·지속적 보호 필요
- 사무처리 능력이 지속적으로 결여된 경우
- 가장 넓은 개입

### 한정후견
- 중요한 사무 전반에 상당한 지원 필요
- 사무처리 능력이 부족한 경우
- 성년후견보다 좁고 특정후견보다 넓음

### 특정후견
- 특정 사무 또는 일시적 후원 필요
- 상속협의, 부동산 매각, 특정 소송, 시설 입소 계약 등 사건 중심
- 자기결정권 제한을 최소화하면서 필요한 보호를 부여

### 후견계약
- 장래 의사결정 저하에 대비한 사전 계약
- 공정증서 필수
- 임의후견감독인 선임 시 효력 발생
- 본인의 의사 반영이 강함

## 4. 실무상 오해 포인트
1. 치매 진단 = 성년후견 아님
2. 특정후견 = 약한 제도 아님
3. 후견계약 = 공정증서 + 감독인 선임이 있어야 작동
4. 한정후견 = 생각보다 종료가 쉽지 않음
5. 후견은 선임으로 끝나지 않고 등기·감독·사무보고가 뒤따름

## 5. 선택 기준 요약
- 전반적·지속적 보호 → 성년후견
- 상당한 보호 필요 → 한정후견
- 특정 문제 해결 → 특정후견
- 미리 대비 → 후견계약

## 6. 작성 가이드
- 정보 수집 → 분석 → 결론 구조 유지
- 상담형 도입부 사용
- 실제 가족이 묻는 질문형 소제목 사용
