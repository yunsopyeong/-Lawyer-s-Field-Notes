# 윤소평 변호사 법률 AI 에이전트 프로젝트

이 저장소는 윤소평 변호사 프로젝트를 기준으로  
대한민국 법률정보, 법률 콘텐츠, 프롬프트, 소스 자료, 운영 문서를 체계적으로 관리하기 위한 작업 저장소입니다.

회생·파산을 중심으로 하되,  
민사, 형사, 가사, 상속, 성년후견 등 관련 생활법률 콘텐츠까지 함께 다룹니다.

---

## 저장소 목적

- 법률 콘텐츠 초안과 최종본을 분리해 관리
- 네이버 블로그, 브런치, 프리미엄콘텐츠, 유튜브 대본 축적
- 프롬프트 및 운영 기준 정리
- 법령, 판례, 실무준칙 소스 관리
- 반복 가능한 콘텐츠 생산 구조 구축

---

## 핵심 원칙

- 대한민국 법령·판례·공식자료 우선
- 정보 수집 → 분석 → 결론 구조 유지
- 과장광고, 단정, 공포 문구 금지
- 상담형 문체 유지
