# -Lawyer-s-Field-Notes
변호사의 사건관련, 재판관련, 의뢰인 관련한 에피소드를 소개합니다. 
# 변호사의 현장노트 | Lawyer’s Field Notes

법률은 조문만으로 끝나지 않습니다.  
상담실에서 시작된 질문, 서면에서 다듬어진 논리, 사건 진행 중 반복해서 확인하게 되는 실무의 기준이 함께 움직입니다.

이 프로젝트는 대한민국 변호사로서 일상적으로 마주하는 질문과 판단, 설명과 기록을 차곡차곡 정리해 두는 실무 아카이브입니다.

## About this project

이곳은 단순한 법률 자료 보관함이 아닙니다.  
실무에서 자주 부딪히는 질문, 의뢰인이 실제로 궁금해하는 지점, 사건을 진행하면서 반복해서 확인하게 되는 체크포인트를 기록하고 구조화하는 공간입니다.

좋은 설명은 단순히 많이 아는 데서 나오지 않고,  
무엇을 먼저 물어야 하는지,  
어디서 오해가 생기는지,  
어떤 자료가 빠지면 절차가 흔들리는지를 아는 데서 나온다고 생각합니다.

그래서 이 저장소는 법률지식만 모으는 곳이 아니라,  
변호사의 실무 감각을 기록하는 작업실에 가깝습니다.

## What I archive here

- 상담에서 반복되는 질문
- 사건 진행 중 확인하는 실무 포인트
- 서면과 설명의 구조
- 블로그와 유튜브로 확장 가능한 콘텐츠 초안
- 다시 사용할 수 있는 체크리스트와 문장
- 법률가의 언어를 일반인의 언어로 바꾸는 설명 방식

## Focus areas

- 기업회생 / 법인회생 / 간이회생 / 일반회생
- 법인파산 / 기업파산 / 개인회생 / 개인파산
- 민사소송 / 보전처분 / 강제집행
- 형사사건 / 사기죄 / 음주운전
- 성년후견 / 상속 / 가사 실무
- 법률 블로그 / 브런치 / 유튜브 스크립트

## Principles

- 법령, 판례, 공식자료를 우선 확인합니다.
- 실무상 오해가 많은 지점을 함께 짚습니다.
- 확인되지 않은 내용은 단정하지 않습니다.
- 복잡한 내용을 가능한 한 분명한 문장으로 정리합니다.
- 일회성 답변보다 반복 가능한 지식으로 남깁니다.

## Why this repository exists

실무 경험은 의외로 빨리 흩어집니다.  
좋은 질문, 좋은 설명, 좋은 판단 기준도 기록해 두지 않으면 다시 처음부터 만들게 됩니다.

이 프로젝트는 그 경험을 남기고, 다듬고, 다시 꺼내 쓰기 위한 저장소입니다.  
결국 더 나은 상담, 더 나은 서면, 더 나은 콘텐츠로 이어지기 위한 기록입니다.

## Note

이 저장소의 내용은 일반적인 법률정보 정리와 실무 경험 기록을 위한 자료입니다.  
구체적인 사건은 사실관계와 자료에 따라 결론이 달라질 수 있습니다.
