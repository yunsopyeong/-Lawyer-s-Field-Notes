# 윤소평 변호사 법률 AI 에이전트 프로젝트

코덱스(Codex)에 맡기기 적합한 업무를 정리한 운영 문서이다.

## 기본 원칙

- 여러 파일을 한 번에 만들거나 수정하는 일
- 저장소 구조를 정리하는 일
- 반복 문서를 자동 생성하는 일
- 검수 기준을 코드나 규칙으로 옮기는 일
- 브랜치 단위 작업과 PR 생성이 잘 맞는 일

반대로 아래 업무는 코덱스보다 채팅형 작업이 더 낫다.

- 법률 설명 자체
- 블로그 글 1편 작성
- 유튜브 대본 1편 작성
- 판례 취지 요약
- 상담형 문장 수정
- FAQ 문장 생성
