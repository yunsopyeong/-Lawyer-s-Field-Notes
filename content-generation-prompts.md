# Content Generation Prompts

## 문서 목적
이 문서는 프로젝트 대화에서 반복적으로 사용한 핵심 프롬프트를 1개 파일에 모은 GitHub 업로드용 프롬프트 묶음이다.

---

## 1. 네이버 블로그 메인 프롬프트

```text
아래 자료를 바탕으로 네이버 블로그 발행용 원고를 작성해 주세요.

[목표]
- 대한민국 법령, 판례, 공식자료를 우선 반영
- 사람이 실제 상담하면서 설명한 글처럼 자연스럽게 작성
- 네이버 SEO를 고려하되 키워드 과다반복과 과장표현은 피하기
- 글 구조는 정보 수집 → 분석 → 결론 순서 유지

[반드시 반영할 사항]
1. 제목 5개 제안
2. 도입부 2개 제안
3. 실무상 오해 포인트 2개 이상 반영
4. 법령, 판례, 공식자료 자연스럽게 반영
5. FAQ 5개 작성
6. 결론은 체크포인트 정리형으로 마무리
7. 내부링크 문구 3개
8. 해시태그 10개

[추가 정보]
- 주 키워드:
- 보조 키워드:
- 독자 유형:
- 검색 의도:
- 반드시 들어갈 법령:
- 반드시 들어갈 판례 또는 취지:
- 반드시 들어갈 공식자료:
- 실무상 꼭 넣을 포인트:
- 실제 상담에서 자주 듣는 질문:
- 내가 강조하고 싶은 결론:
- 피하고 싶은 표현:
- 참고 자료 원문:
```

## 2. 브런치 메인 프롬프트

```text
아래 자료를 브런치 발행용 글로 작성해 주세요.

[목표]
- 블로그보다 조금 더 서사적이고 읽히는 흐름
- 그러나 대한민국 법률 기준의 정확성 유지
- 독자가 스스로 판단할 수 있도록 질문과 설명을 교차 배치
- 글 구조는 정보 수집 → 분석 → 결론 순서 유지

[반드시 반영할 사항]
1. 제목 5개 제안
2. 오프닝 2개 제안
3. 실무상 주의점 3개 이상 반영
4. 법령 또는 판례 취지를 설명형 문장으로 반영
5. 광고형 문장 대신 판단 포인트 중심으로 마무리
```

## 3. 네이버 프리미엄콘텐츠 프롬프트

```text
아래 자료를 네이버 프리미엄콘텐츠 발행용 원고로 작성해 주세요.

[목표]
- 무료 블로그 글보다 더 깊은 설명 제공
- 체크리스트, 예외, 리스크, 서류 포인트 포함
- 실제 의사결정에 도움이 되는 정보 밀도 유지

[반드시 반영할 사항]
1. 제목 5개 제안
2. 이 글이 필요한 독자 유형 명확히 제시
3. 체크리스트와 예외 상황 포함
4. 법령, 판례, 공식자료, 통계 중 최소 2종 이상 반영
5. 결론은 실행 체크리스트로 마무리
6. 연계 읽기 3개 제안
```

---

## 4. 유튜브 대본 프롬프트

```text
아래 자료를 유튜브 촬영용 설명형 대본으로 작성해 주세요.

[목표]
- 변호사가 카메라 앞에서 차분히 설명하는 말투
- 처음 15초~30초 안에 시청자가 끝까지 볼 이유 제시
- 실제 상담 질문을 구조에 반영
- 정보 수집 → 분석 → 결론 흐름을 영상 구조에 맞게 적용

[반드시 반영할 사항]
1. 제목 5개
2. 썸네일 문구 10개
3. 오프닝 훅 3개
4. 전체 대본
5. 클로징 문구 3개
6. 설명란 초안
```

---

## 5. 초압축 호출문

### 블로그용
- 아래 자료를 반영해 네이버 블로그 발행형으로 작성해 주세요. 정보 수집 → 분석 → 결론 구조, 대한민국 법령·판례·공식자료 우선, FAQ와 해시태그까지 포함해 주세요.
- 아래 키워드로 대표이사 또는 채무자 관점의 네이버 블로그 글을 써 주세요. 상담형 도입부, 실무상 오해 포인트, 결론 체크리스트를 꼭 넣어 주세요.

### 유튜브용
- 아래 자료를 바탕으로 말하기 쉬운 유튜브 촬영 대본을 작성해 주세요. 도입 15초 후킹, 상담형 말투, 실무상 오해 포인트 포함이 필요합니다.
- 아래 키워드로 롱폼 유튜브 대본과 썸네일 문구, 설명란까지 한 번에 작성해 주세요. 구조는 도입 → 핵심 설명 → 오해 포인트 → 정리 순서입니다.

### 프리미엄용
- 아래 자료를 바탕으로 네이버 프리미엄콘텐츠용 체크리스트 강화본을 작성해 주세요. 예외, 리스크, 서류 포인트까지 넣어 주세요.
